# Прокси для Gemini Live API через Cloudflare Workers

Этот воркер нужен пользователям из стран, где Google AI API заблокирован
(Беларусь, Россия, Иран, Куба и пр.). Он принимает WebSocket-соединение от
браузера Liv и проксирует его в `generativelanguage.googleapis.com` — для
Google запрос выглядит как «из США/Европы», и Беларусь/Россия Google не блокирует.

## Деплой

1. Зарегистрируйся на [dash.cloudflare.com](https://dash.cloudflare.com/sign-up)
   (бесплатно, без карты).
2. **Workers & Pages → Create Worker → Start with Hello World!**
3. Назови воркер (например `livvv-proxy`) → **Deploy**.
4. Открой воркер → **`</> Edit code`** → удали дефолтный код, вставь
   содержимое [`cloudflare-worker.js`](./cloudflare-worker.js) → **Deploy**.
5. Cloudflare покажет URL воркера, например
   `https://livvv-proxy.<your-subdomain>.workers.dev`.
6. Проверь: открой `https://livvv-proxy.<your-subdomain>.workers.dev/health`
   в браузере — должно вернуться `Liv proxy is alive...`.

## Использование в Liv

В настройках Liv → раздел «Маршрут (для стран с блокировкой)»:

- Включи переключатель «Подключаться через прокси».
- В поле «Адрес прокси» вставь `livvv-proxy.<your-subdomain>.workers.dev`
  (без `https://`, но если вставишь с протоколом — он автоматически уберётся).

Перезапусти сессию. В журнале событий должна появиться строка
«Соединение через прокси: ...».

## Лимиты

Бесплатный тариф Cloudflare Workers: **100 000 запросов в день**, что для
персонального использования никогда не закончится. Каждое сообщение Live API
(аудиочанк, текстовая фраза) — это не отдельный «запрос» с точки зрения
Cloudflare, потому что мы всё гоним через одну долгую WebSocket-сессию.

## Безопасность

Воркер — публичный URL без аутентификации. Если кто-то узнает адрес твоего
воркера, он сможет ходить через него к Gemini API со своим API-ключом.
Это не утечка твоего ключа (ключ остаётся в браузере у каждого
пользователя), но это «бесплатный туннель», который ты раздаёшь.

Если хочется ограничить — добавь в воркер проверку секретного заголовка или
конкретного origin (например `*.vercel.app`). Для текущей задачи (личный
проект, который иногда показываешь друзьям) — это нестрашно.
